# Tech Stack

## Core Technologies
- **SCSS/Sass**: Primary styling language (`sass ^1.77.2`)
- **CSS3**: Modern CSS features with flexbox and grid support
- **DocFX**: Documentation generation system
- **Node.js/npm**: Package management and build tools

## Build Tools
- **Sass**: SCSS compilation to CSS
- **PostCSS**: CSS post-processing with cssnano for minification
- **Nodemon**: Development file watching
- **Browser-sync**: Live reload development server
- **npm-run-all**: Parallel script execution

## Testing
- **Sass-True**: SCSS unit testing framework (`sass-true ^8.0.0`)
- **Mocha**: Test runner (`mocha ^10.4.0`)

## Development Dependencies
- **cpy-cli**: File copying utility
- **rimraf**: Cross-platform file/directory removal
- **prettier**: Code formatting
- **cssnano**: CSS minification
- **postcss-cli**: PostCSS command line interface

## Architecture
- **SCSS Modules**: Using `@use` and `@forward` for modern SCSS architecture
- **Component-based**: Organized by components, elements, forms, layouts, etc.
- **CSS Custom Properties**: Modern CSS variables implementation
- **Responsive-first**: Mobile-first responsive design approach
