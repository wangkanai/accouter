# Project Purpose

**Accouter** is a semantic and open-source administrative template framework built with SCSS/CSS.

## Key Features
- **Admin Template Framework**: Pre-built components for creating administrative interfaces
- **Responsive Design**: Compatible with all modern browsers and devices
- **CSS Framework**: Built on SCSS with modern CSS3 features
- **Documentation Integration**: Uses DocFX for documentation generation
- **Component Library**: Over 10 demo pages with customizable components using Sass

## Target Audience
- Developers creating admin panels and administrative interfaces
- Projects needing clean, professional UI components
- Teams wanting to quickly build functional interfaces with minimal custom CSS

## Distribution
- Distributed via npm as `@accouter/core`
- Main entry point: `accouter.scss`
- Compiled CSS available as `css/accouter.css` and minified version
- Preview available at: https://preview.accouter.dev/
- Documentation at: https://accouter.dev/docs
